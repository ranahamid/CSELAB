So I figured out the issue.

in the line:

view source
print?
1os.print("User Name: ");
2String checkUser = in.readLine();


the readLine function doesn't stop reading unless terminated by a \n or \r and possibly another, I can't remember. So the buffered reader was still trying to read before sending out the new text.

Fixed it by changing the print to a println.

Simple fix, but I might fool around with it to get the output I want.

Thanks for your help though!
