Hamming Code

This week, in our Networking Lab, we were asked to create a C++ program which would generate the Code Word of a given message. I didn’t even bother opening my terminal… And, since my first interaction with hamming code in my Digital Communication Lab didn’t start off well, I never had any intentions of completing it.

However, after 2-3 days of research and strict coding, I was able to make the Hamming Code program.

I started referring these two sites ( extremely helpful )

  • Hamming Code Wiki
  • Calculating Hamming Code

I had used odd parity and my parity bits started from the end of the code ( p4 _ _ _p3 _ p2 p1 )

Since, hamming code is more or less related to bits, I had googled few topics on bits manipulation in C++. My program included 3 functions and a main function :

  • no_of_parity_bits()
  • codeWordGeneration()
  • finalCodeWord()

#define isPowerOfTwo(x) (!((x-1) & x))

The following function calculates number of parity bits required for a message.

int no_of_parity_bits( int msg_length )
{

int i = 2;
int p = 2;

while ( ( pow(2,i) - 1 - p ) < msg_length ) {

i++;
p++;

}

return p;
}

The following generates a temporary code word keeping parity bits as 0. For eg, if message is 3 ( i.e. 11 in binary ) it would generate a code word 10100(in binary).

int codeWordGeneration ( int msg, int msg_size )
{
int temp;
int j = 1;
int CW = 0;
int i = 0;

while ( i > i) & 1 );
CW = ( CW | ( temp << ( j -1 ))) ;
i++;
}
j++;

}

return CW;
}

And finally, the follwing function returns the final code word.

int finalCodeWord( int msg, int msg_size, int parity_bits )
{
int i = 0;
int j;
int m;
int k;
int l = 2;
int parity = 0;
int n;

while ( i < parity_bits ) {

m = 1 << i;
j = m;
parity = 0;

while ( j > (j-1)) & 1 );

while ( ( k = ( msg_size + parity_bits ) )
break;
n++;
parity = parity ^ ( msg >> (n-1 ) & 1 );
k++;

}

j = j + l;

}

if ( parity == 0 ) msg |= ( 1 << (m-1) );
i++;
l = pow(2,i+1);

}

return msg;
}

My program required two User inputs, One for message (in decimal ) and the other for size of message in binary. For e.g. if msg was 3, then msg size must be 2 ( 3 ==> 11 in binary ).

int main()
{
int msg;
int msg_size;

cin >> msg;
cin >> msg_size;

cout << finalCodeWord( codeWordGeneration( msg, msg_size ), msg_size, no_of_parity_bits( msg_size ) );

return 0;
}

Nevertheless, it was fun to code the thing i didn’t want to… Learnt some awesome usage of bits manipulation, didn’t use any data structures..
